Manual de Usuario
Funcionalidades Principales
Dibujar Figuras Geométricas: El programa te permite dibujar diferentes figuras geométricas como triángulos, cuadrados, rectángulos, círculos, líneas, rombos y hexágonos.
Personalización: Puedes personalizar el color del texto y el carácter con el que se dibujan las figuras.
Limpiar Pantalla: Borra el contenido de la pantalla para iniciar un nuevo dibujo.
Grabar Pantalla: Guarda el contenido actual de la pantalla en un archivo de texto.
Abrir Archivo: Abre un archivo de texto y muestra su contenido en pantalla.
Uso Básico
Selección de Figuras: Presiona las teclas de función correspondientes (F1 a F7) para seleccionar la figura que deseas dibujar.
Personalización: Utiliza las teclas de función F8 para cambiar el carácter de dibujo y F10 para cambiar el color del texto.
Limpiar Pantalla: Presiona la tecla de función F9 para limpiar la pantalla y comenzar un nuevo dibujo.
Grabar y Abrir Archivo: Utiliza las combinaciones de teclas Ctrl + A para abrir un archivo y Ctrl + E para grabar el contenido de la pantalla.

Manual Técnico
Requisitos del Sistema
Windows OS
Compilador de C++
Librerías Utilizadas
<iostream>: Para entrada y salida estándar.
<conio.h>: Para el uso de funciones de control de la consola.
<windows.h>: Para acceder a funciones de manipulación de la consola.
<cmath>: Para operaciones matemáticas.
<string>: Para el manejo de cadenas de texto.
<fstream>: Para operaciones de lectura y escritura de archivos.
<vector>: Para la gestión de vectores dinámicos.
Funciones Principales
Dibujo de Figuras: Funciones como dibujarTriangulo, dibujarCuadrado, dibujarRectangulo, dibujarCirculo, dibujarLinea, dibujarRombo y dibujarHexagono son responsables de dibujar las diferentes figuras geométricas en la consola.
Personalización: Las funciones seleccionarColor y seleccionarCaracter permiten al usuario personalizar el color del texto y el carácter de dibujo respectivamente.
Manipulación de Pantalla: Funciones como limpiarPantalla, grabarPantalla y abrirArchivo gestionan la limpieza de la pantalla, la grabación de su contenido en archivos y la apertura de archivos existentes.
Interacción de Usuario: El programa utiliza la biblioteca conio.h para capturar las teclas presionadas por el usuario y responder en consecuencia.
